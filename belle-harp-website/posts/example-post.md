# Example Blog Post

This is a sample blog post written in Markdown. When you set up a static site generator like Astro or Next.js, posts like this will be automatically converted to HTML and displayed on your blog page.

## Features

- **Markdown Support**: Write your posts in Markdown for easy formatting
- **Frontmatter**: Use YAML frontmatter to add metadata like title, date, tags, etc.
- **Automatic Processing**: Static site generators can automatically process these files

## Getting Started

1. Create a new `.md` file in the `posts` directory
2. Add frontmatter with post metadata
3. Write your content in Markdown
4. The static site generator will process it and add it to your blog

## Next Steps

To fully set up your blog with Markdown support:

1. Choose a static site generator (Astro, Next.js, or similar)
2. Configure it to read from the `posts` directory
3. Set up a build process to convert Markdown to HTML
4. Update `blogs.html` to fetch and display posts dynamically

Happy blogging! 🎉
